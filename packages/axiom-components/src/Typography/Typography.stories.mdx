## Text Style

Strong can be used sparingly when extra emphasis is needed.

<Preview>
  <Strong>Strong text</Strong>
</Preview>

Disabled is used exclusively to indicate a non-interactive state

<Preview>
  <Text textColor="disabled">Strike text</Text>
</Preview>

Accent is used exclusively to indicate an interactive state such as links.

<Preview>
  <Text textColor="ui-accent">accent text</Text>
  <Link>A link</Link>
</Preview>

Success is used exclusively to indicate an valid state

<Preview>
  <Text textColor="success">success text</Text>
</Preview>

Error is used exclusively to indicate an invalid state

<Preview>
  <Text textColor="error">error text</Text>
</Preview>
