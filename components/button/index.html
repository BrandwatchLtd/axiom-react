Button group

ButtonGroups keep a consistent flow of spacing between Buttons and sibling elements, so liek other inline elements they should always be used, even when there is only one button.

  • React
  • Raw HTML
<ButtonGroup>
  <Button> ... </Button>
  <Button> ... </Button>
  <Button> ... </Button>
</ButtonGroup>
