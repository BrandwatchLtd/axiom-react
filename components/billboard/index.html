Billboard

Base Billboard

Takes one of the palette or context colour ids as well as a variation [optional].

I am a billboard.
  • React
  • Raw HTML
<Billboard color="grey" variation="900"> ... </Billboard>

Image Billboard

I am a billboard with an image!
  • React
  • Raw HTML
<Billboard color="grey" image="/assets/axiom-bg.jpg" variation="900"> ... </Billboard>

Overlay Billboard

I am a billboard with an overlay!
  • React
  • Raw HTML
<Billboard color="grey" image="/assets/axiom-bg.jpg" overlay={true} variation="900"> ... </Billboard>
