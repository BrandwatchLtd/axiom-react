Packages

Core packages

The most common and core packages to help build Brandwatch applications. Made up of React components and design resources.
@brandwatch/axiom-materials

Low level design resources that are used to make up Axiom components and composites (animations, colors, sizes etc...).

@brandwatch/axiom-components

Core Axiom components package, includes the main components that make up a Brandwatch application.

@brandwatch/axiom-composites

Composites are reusable React components but designed to be used within a single context of an a Brandwatch application.

@brandwatch/axiom-charts

Charting components, including low level components that make up charts.

Supplementary packages

Additional packages used internally within Axiom, but are still useful within applications.
@brandwatch/axiom-automation-testing • Awaiting docs

Automation testing attribute selector and IDs for elements of interest inside React component templates.

@brandwatch/axiom-localization • Awaiting docs

Formatting utils for providing consistent data formats. Like dates and numbers.

@brandwatch/axiom-formatting • Awaiting docs

Formatting utils for providing consistent data formats. Like dates and numbers.

@brandwatch/axiom-utils • Awaiting docs

Formatting utils for providing consistent data formats. Like dates and numbers.

@brandwatch/babel-plugin-axiom-imports • Awaiting docs

An Axiom specific babel plugin that transforms and removes unused Axiom exports.

@brandwatch/babel-plugin-transform-svg-axiom • Awaiting docs

An Axiom specific babel plugin that inlines, optimises and recolours SVGs for Brandwatch applications.

Development workflow packages

Packages that help improve development workflows.

Webpack loader for Hot Module Reloading React component API docs, generated from PropTypes.

@brandwatch/axiom-documentation-viewer • Awaiting docs

Axiom React component documentation viewer for Style Guide Driven Development. Provides component preview and PropTypes configuration table.

Eslint config for some Axiom Brandwatch applications.

@brandwatch/stylelint-config-axiom • Awaiting docs
